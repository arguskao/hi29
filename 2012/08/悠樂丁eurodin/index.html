悠樂丁Eurodin

Aug 6, 2012 in 安眠藥
bzd 失眠
2 min read

Eurodin是一個歷史相當悠久，而且知名度也相當高的精神科藥物，它是BZD類的安眠藥，屬於中效型的，和Diazepam，Nitrazepam等Benzodiapine類化合物具相同藥理作用(如鎮靜，馴化，導眠，增強睡眠，抗痙攣及肌肉鬆弛等作用)特別對於睡眠，作用強且安定。本品之催眠機轉與Benzodiazepine類化合物相同，被認為是對大腦邊緣及下視丘腦幹維持機制的抑制，並非直接作用於覺醒機制，因為不至於導致麻醉作用。

Eurodin 中文名稱悠樂丁，成分為 estazolam

適應症：失眠

Eurodin 經口服吸收投與後，能迅速的經由胃腸道吸收，雖然目前尚無人體之生體可用率報告，動物試驗的生體可用率高達98%，食物對於Eurodin的影響尚未確立。Eurodin的半衰期約為24小時。 副作用：

  1. 突然中止投藥會有譫妄，痙攣等症狀。
  2. 對精神分裂症狀投與本劑反而導致刺激，興奮，錯亂等奇異反應。
  3. 對肝臟偶有GOT，GPT上升現象。
  4. 對腎臟BUN上升現象。
  5. 偶有貧血，白血球過低的現象。

注意事項：

  1. 肺性心肺氣腫，支氣管氣喘等嚴重呼吸功能不足者應小心使用
  2. 使用時宜避免開車或操作危險機械
  3. 服藥期間最好避免飲酒
  4. 不可併服葡萄柚或葡萄柚汁時
  5. 患有心臟、肝臟、腎臟功能障礙及腦器質障礙患者應該依照醫師指 示，謹慎服用本藥。
